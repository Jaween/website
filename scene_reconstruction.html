Scene Reconstruction
2015 March to June

This is a partial system for reconstructing a 3D model based on stereo video captured from a pair of RGB cameras. It specifically avoids the use of an infrared depth camera. The code was hardware accelerated using OpenCL and followed the outline presented in the KinectFusion paper.

This project was a primarily self-directed as part of my undergraduate research course ‘Advanced Topics in Computer Science’, with Professor Ian Reid providing additional guidance throughout.
