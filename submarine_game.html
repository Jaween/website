Submarine Game
2017 January

Developed in Unity over 48 hours as part of the 2017 Global Game Jam. The theme of this jam was 'Waves'. The player navigates a pitch black tunnel using a puling sonar to find their way through.

The primary contribution I made was the sonar effect, which was developed as a modification to Unity's standard shaders. The effect is achieved by representing the sonar wavefront as a plane which travels down the tunnel. Each fragment of the tunnel geometry is shaded based on their distance to that plane. So the closer the fragment is to the wavefront, the brighter it is.

This logic was inserted into the standard Unity shader, so that the artists could take advantage of the existing shader functionality. But my lack of experience with Unity's shader pipeline (and shaders in general) resulted in faint square-shaped artifacts around lights.
